##  Overview
This project demonstrates automated **REST API testing** using:
-  **RestAssured**
-  **JUnit 5**
-  **Java 17**
-  **Maven**
-  **Jenkins CI/CD**
-  **Docker (optional setup)**

The tests validate API endpoints by sending HTTP requests and verifying responses, status codes, and JSON payloads.
